## Run Locally

Clone the project

```bash
  git clone https://github.com/INeedJobToStartWork/LinkShorter.git
```

Go to the project directory

```bash
  cd LinkShorter
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
